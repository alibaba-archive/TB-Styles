Overview

在 CSS Style Guide 中提到关于 Variable 的建议：

  • Mixin 与 Variable 采用驼峰命名法
  • Variable 名字使用 $ 作为前缀

框架中充满大量（50 个）全局的 Mixin 与 Variable，是应该被反对的
如果想让变量具有语义，欢迎放到局部作用域中
